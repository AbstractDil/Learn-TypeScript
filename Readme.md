
# Typescript

TypeScript is JavaScript with syntax for types.

# Commands

```bash 
tsc --init # Create tsconfig.json
tsc # Compile all .ts files
tsc --watch # Compile all .ts files and watch for changes
```
# Use of noEmitOnError

```json
{
  "compilerOptions": {
    "noEmitOnError": true
  }
}
```
